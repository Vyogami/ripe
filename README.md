# Vyogami's Wiki

My personal wiki on various topics, tools, fields.

## Contributing

1. Install dependencies:

   ```shell
   bun install
   ```

1. Start the development server:

   ```shell
   bun start
   ```

1. Check for formatting problems:

   ```shell
   bun check
   ```

1. Format the codebase:

   ```shell
   bun format
   ```
